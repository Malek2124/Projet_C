# Projet_C
Module C/GTK pour gérer les entraîneurs dans une app de gestion de salle de sport. L’admin peut ajouter, modifier, supprimer, rechercher et afficher les entraîneurs. Chaque coach peut s’inscrire à un cours sportif selon sa spécialité et disponibilité. Interface Glade, stockage CSV, champs complets (nom, email, tel, sexe, etc.).
